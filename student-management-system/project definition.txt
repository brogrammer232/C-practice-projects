
	Student management system.
	==========================
Concept:
	Store and manage student records using structures.

Features:
	Store the following student details:
		Name.
		Student ID.
		Grade.
		Address:
			Town.
			Street.
			Zip.
	Add new students.
	Display student details.
	Sort students by grade.

Practice concepts:
	Nested sturctures.
	Arrays of structures.
	File handling (optional).
	Sorting and searching.

Bonus challenge:
	Save and load data from a file.
	Implement a ranking system based on marks.
